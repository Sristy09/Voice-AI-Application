# 🎙️ Voice AI Application

## 📌 Description
This is a Voice AI application that:
- Records audio from microphone or uses an audio file
- Converts speech to text (ASR)
- Generates a response using simple AI or rule-based logic
- Converts response to speech (TTS)
- Plays or saves output audio

## 💡 Use Case
Language Practice Tool – helps users practice speaking and get responses.

## ⚙️ Technologies Used
- Python
- SpeechRecognition
- pyttsx3 / gTTS

## ▶️ How to Run

1. Clone repository:
git clone https://github.com/Sristy09/Voice-AI-Application.git

2. Install requirements:
pip install -r requirements.txt

3. Run:
python main.py
